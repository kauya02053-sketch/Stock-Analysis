# 📈 股票自動化分析工具 (Stock Analysis Tool)

這是一個使用 Python 自動抓取即時股市數據並進行分析的專案。

### 🚀 功能特點
* **即時數據：** 使用 `yfinance` 串接 Yahoo Finance API。
* **技術指標：** 自動計算 20 日移動平均線 (MA20)。
* **自動決策：** 根據價格與均線關係，給予「看多」或「看空」建議。

點擊下方的按鈕，即可在瀏覽器中直接執行分析：
